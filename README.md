Clean an e-commerce dataset, perform KPI analysis, generate business insights, and create visualizations.

- NumPy
- Matplotlib
- Seaborn

## Tasks Completed

### Data Cleaning
- Handled missing values
- Removed duplicate rows
- Standardized date formats
- Removed invalid records
- Created cleaned dataset

### KPI Analysis
- Total Revenue
- Total Orders
- Average Order Value
- Total Units Sold
- Top Product Analysis

### Visualizations
- Monthly Revenue Trend
- Customer Revenue Share
- Top Selling Products
- Dashboard Mockup

## Key Findings
- Registered customers contributed 83.5% of revenue.
- Revenue peaked in November.
- PAPER CRAFT, LITTLE BIRDIE was the most demanded product.
- DOTCOM POSTAGE generated the highest revenue.
